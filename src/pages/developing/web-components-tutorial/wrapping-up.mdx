## Badge information

For more information about the IBM Carbon Design System Developer Essentials -
Carbon for IBM.com Web Components badge, see the [Acclaim](https://www.youracclaim.com/org/ibm/badge/tbd)
site.

### Badge process

This badge demonstrates knowledge about Carbon for IBM.com Web Components. To earn the
badge:

1. Complete steps 1 through 5 of the Carbon for IBM.com Web Components tutorial:
   - Step 1. [Installing Carbon](/developing/web-components-tutorial/step-1)
   - Step 2. [Adding out of the box components](/developing/web-components-tutorial/step-2)
   - Step 3. [Adding custom Content block components](/developing/web-components-tutorial/step-3)
   - Step 4. [Adding custom Carbon components](/developing/web-components-tutorial/step-4)
   - Step 5. [Deploying to GitHub Pages](/developing/web-components-tutorial/step-5)
2. Have approved pull requests for steps 1 through 5 in the
   [carbon-for-ibm-dotcom-tutorial repository](https://github.com/carbon-design-system/carbon-for-ibm-dotcom-tutorial).
3. Apply for the IBM Digital Badge by completing the [form and survey](https://badges.carbondesignsystem.com/).
