## Modular packages

In addition to the framework packages (<Link to="/developing/frameworks/web-components">Web Components</Link>, <Link to="/developing/frameworks/react">React</Link>, and <Link to="/developing/frameworks/react-wrapper">React wrapper</Link>)
there are other public packages available that are used within the framework packages, and could also be used as
isolated modular functionality and styling within an application.

### Styles

In order to share styles across the framework packages, styles have been separated into its own standalone package.

Carbon for IBM.com contains its own API and utility functionality that have been separated out into their own modular
packages. They are exported as pure ES6 classes.
