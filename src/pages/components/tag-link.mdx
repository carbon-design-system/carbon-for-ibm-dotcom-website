## Overview

Tag link is a clickable tag that links the topic to another page for users to find more information about the topic. It uses Carbon's [Tag](https://carbondesignsystem.com/components/tag/usage/) as a base, and uses a larger type token and additional spacing around the label. Unlike Tag, Tag link has set color styles for the label and background.

Tag link is primarily used in an optional group in the [Lead space](https://www.ibm.com/standards/carbon/components/leadspace). Used in this way, the Tag links add categories and additional context to the Lead space.
