## Overview

Logo grid consists of a heading and a group of logos. Each logo should be contained inside a space (4:3 aspect ratio) and the logos are not clickable.

![default](../..//images/pattern/logo-grid/logo-grid-default.png)

<br />

## Logo placement

Logos should be centered within a 4:3 aspect ratio to create optical alignment, visual hierarchy, and ensure all brands have equal representation.

### Grid template

Use the [logo grid template](https://ibm.box.com/s/t85vh5blf8lbya13ro2c40pzp49m5es1) to size logos and align them within the varying grid guides to achieve equal visual hierarchy.

### Tips and techniques

#### Alignment
Logos should be vertically and horizontally centered within a 4:3 aspect ratio. Consider the weight, width, and height when aligning the logo to the grid to find equal weight with all logos. Don’t fit every logo to the max width.

<DoDontRow>
  <DoDont caption="Always place logo in the center of the template and scale to fit the height or width within the logo grid guides.">

![Always place logo in the center of the template and scale to fit the height or width within the logo grid guides.](../../images/pattern/logo-grid/logo-grid-alignment-do.png)

  </DoDont>
  <DoDont type="dont" caption="Don't scale every logo to the max width.">

![Don't scale every logo to the max width.](../../images/pattern/logo-grid/logo-grid-alignment-dont.png)

  </DoDont>
</DoDontRow>

#### Orientation
When available, use horizontally-oriented logos to fit comfortably within the grid and to maintain visual hierarchy.

<DoDontRow>
  <DoDont caption="Horizontally oriented logos are easier to fit within the grid template to maintain optical hierarchy with other logos.">

![Horizontally-oriented logos are easier to fit within the grid template to maintain a consistent hierarchy with other logos.](../../images/pattern/logo-grid/logo-grid-orientation-do.png)

  </DoDont>
  <DoDont type="dont" caption="Avoid using vertically oriented logos that don’t fit nicely within the grid template.">

![Avoid using vertically oriented logos that don’t fit nicely within the grid template.](../../images/pattern/logo-grid/logo-grid-orientation-dont.png)

  </DoDont>
</DoDontRow>

#### Type
Use logos that include the company name or wordmark, rather than an icon, to ensure brand recognition.

<DoDontRow>
  <DoDont caption="Use logos with the brand name or wordmark.">

![Use logos with the brand name or wordmark.](../../images/pattern/logo-grid/logo-grid-type-do.png)

  </DoDont>
  <DoDont type="dont" caption="Avoid using icon-only logos to ensure brand recognition.">

![Avoid using icon only logos to ensure brand recognition.](../../images/pattern/logo-grid/logo-grid-type-dont.png)

  </DoDont>
</DoDontRow>

#### Color
When placed in a group, ensure visual consistency by using either all full-color logos or all monotone logos.  Don’t mix full color and monotone, unless the brand colors are black and white.

<DoDontRow>
  <DoDont caption="Do use full color logos when placed in a group.">

![Do use full-color logos for all logos placed in a group.](../../images/pattern/logo-grid/logo-grid-color-do.png)

  </DoDont>
  <DoDont type="dont" caption="Don't mix full-color and monotone logos when placed in a group.">

![Don't mix full-color and monotone logos when placed in a group.](../../images/pattern/logo-grid/logo-grid-color-dont.png)

  </DoDont>
</DoDontRow>
