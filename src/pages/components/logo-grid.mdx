## Default

Logo grid consists of a heading and a group of logos. Each logo should be contained inside a space (4:3 aspect ratio) and the logos are not clickable.

![default](../..//images/pattern/logo-grid/logo-grid-default.png)

<br />

## Placement

Logos should be ceneterd within a 4:3 aspect ratio to create optical alignment, visual hierarchy, and ensure all brands have equal representation.

### Grid template

Use the logo grid template to size logos and align them within the varying grid guides to achieve equal visual hierarchy.

The grid is meant to be used as a guideline. Make optical adjustments when necessary.

#### Alignment
Logos should be vertically and horizontally centered within a 4:3 aspect ratio. Consider the weight, width and height when aligning the logo to the grid to find equal weight with all logos. Don’t fit every logo to the max width.

### Tips and techniques

#### Orientation
Use horizontally oriented logos when possible to fit comfortably within the grid and easily maintain visual hierarchy.

<DoDontRow>
  <DoDont caption="Horizontally oriented logos are easier to fit within the grid template to maintain optical hierarchy with other logos.">

![Horizontally oriented logos are easier to fit within the grid template to maintain optical hierarchy with other logos.](../../images/component/logo-grid/logo-grid-orientation-do.png)

  </DoDont>
  <DoDont type="dont" caption="Avoid using vertically oriented logos that don’t fit nicely within the grid template.">

![Avoid using vertically oriented logos that don’t fit nicely within the grid template.](../../images/component/logo-grid/logo-grid-orientation-dont.png)

  </DoDont>
</DoDontRow>

#### Type
Use logos that include the company name or wordmark, rather than an icon, to ensure brand recognition.

<DoDontRow>
  <DoDont caption="Use logos with the brand name or wordmark.">

![Use logos with the brand name or wordmark.](../../images/component/logo-grid/logo-grid-type-do.png)

  </DoDont>
  <DoDont type="dont" caption="Avoid using icon only logos to ensure brand recognition.">

![Avoid using icon only logos to ensure brand recognition.](../../images/component/logo-grid/logo-grid-type-dont.png)

  </DoDont>
</DoDontRow>

#### Color
Use all full colored or all monotone logos when placed in a group. Don’t mix full color and monotone, unless the brand colors are black and white.

<DoDontRow>
  <DoDont caption="Do use full color logos when placed in a group.">

  </DoDont>
</DoDontRow>
