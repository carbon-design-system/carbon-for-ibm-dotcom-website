## Default
When designed and implemented effectively, the Lead space &mdash; block can be used to engage visitors immediately in one of the most important moments in the user’s journey through a complex web environment.

It includes a headline, subheadline, supporting message, [Link list](https://ibmdotcom-react.mybluemix.net/?path=/story/components-linklist--default) and a call to action. There is also the option to add media in the form of an image or video.

![Default](../../images/pattern/lead-space-block/leadspace-block.jpg)

## Content guidance

| Field | Field Type | Required   | Cardinality | Max character size (English / translated)     |
| -------- | --------- | ---------- | ---------- | ----------|
| Headline | Short Copy | Yes   | 1 | XXS (40 / 55)|
| TOC Headline Override  | Short Copy | No     | 1 | Mini (25 / 35) |
| Sub-Headline  | Short Copy | Yes   | 1 | XS (65 / 85) |
| Paragraph  | Long Copy | Yes     | 1 | XL (250 / 325)   |
| Video or Informative Image | Media | No | 1 | - |
| Link List (End of Section) | Component | No | 1 | - |
| CTA (Button) | Component | No | 1 | - |
