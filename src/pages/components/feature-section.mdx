## Overview

Feature section is a full-width section designed to create a visual break from the rest of the page and focus the viewer's attention on a single story. It contains expressive imagery and a CTA linking to more details. Feature section is intended to be used sparingly to draw attention – avoid stacking multiple feature sections in a row or including too many on a single page.

## Modifiers

### Color schemes

Feature section has additional color schemes to enable more flexibility, these include the default `regular` color scheme, as well as `inverse`, `purple` , and `cyan`.

Note that the color scheme effects the background color adjacent to the imagery, choose a color that supports your image, and avoid using illustrations that have a `10` value from the [IBM Design Language color pallet](https://www.ibm.com/design/language/color#the-palette) (ie `blue 10`, `green 10`, etc).

## Behaviors

### Desktop

The feature section spans the full width of the 16 column grid, and therefore it cannot be used in combination with the vertical table of contents component, utilize the [horizontal table of contents](../components/table-of-contents#horizontal) instead.

### Media

The image maintains a 1:1 aspect ratio across all breakpoints.
