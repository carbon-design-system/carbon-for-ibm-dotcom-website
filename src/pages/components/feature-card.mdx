## Default

Feature card is used for highlighting a piece of content on page. It should be used sparingly to be effective. Feature card has two sizes. The default is medium, and consists of an image, heading, and CTA (call-to-action). Feature card -- large allows additional copy (eyebrow and copy) and occupies 12 columns of the page.

## Content guidance

### Feature card -- medium

| Field                                                                   | Field type  | Required | Cardinality | Max character size (English / translated) |
| ----------------------------------------------------------------------- | ----------- | -------- | ----------- | ----------------------------------------- |
| Heading                                                                 | Short copy  | Yes      | 1           | XS (65 / 85)                              |
| Image                                                                   | Media       | Yes      | 1           | –                                         |
| URL                                                                     | Link        | Yes      | 1           | –                                         |
| [Type](https://www.ibm.com/standards/carbon/components/cta) (ex. Local) | Option list | Yes      | 1           | –                                         |


### Feature card -- large

| Field                                                                   | Field type  | Required | Cardinality | Max character size (English / translated) |
| ----------------------------------------------------------------------- | ----------- | -------- | ----------- | ----------------------------------------- |
| Eyebrow                                                                 | Short copy  | No       | 1           | Mini (25 / 35)                            |
| Heading                                                                 | Short copy  | Yes      | 1           | XS (65 / 85)                              |
| Copy                                                                    | Long copy   | No       | 1           | LL (200 / 260)                            |
| Image                                                                   | Media       | Yes      | 1           | –                                         |
| URL                                                                     | Link        | Yes      | 1           | –                                         |
| [Type](https://www.ibm.com/standards/carbon/components/cta) (ex. Local) | Option list | Yes      | 1           | –                                         |


To see the list of all character count standards, go to this <a href="https://github.com/carbon-design-system/carbon-for-ibm-dotcom-website/wiki/Character-count-standards" target="_blank">link</a>.
