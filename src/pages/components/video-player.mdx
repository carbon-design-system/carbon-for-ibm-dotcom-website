## Default
The Video player aspect ratio should match that of the video, or use one of the standard aspect ratios. In most cases 16x9, or 1x1 is recommended.

## Content guidance

| Field | Field type | Required   | Cardinality | Notes / Options |
| -------- | --------- | ---------- | ---------- | ----------|
| Video ID | Short copy | Yes | 1 | - |
| Autoplay | True/False | No | 1 | True = Autoplay<br/>False = Don't autoplay (default) |
| Aspect ratio | Option list | No | 1 | Default is 16x9. Other ratios are 9x16, 2x1, 1x2, 4x3, 3x4, 1x1 |

To see the list of all character count standards, go to this <a href="https://github.com/carbon-design-system/carbon-for-ibm-dotcom-website/wiki/Character-count-standards" target="_blank" rel="noreferrer noopener nofollow">link</a>.
