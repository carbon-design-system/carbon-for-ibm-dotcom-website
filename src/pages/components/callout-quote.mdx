## Default

The Callout quote pattern uses high contrast colors to create a delightful pause in the narrative and create a powerful impression on the user.

## Content guidance

| Field                                                                                      | Field type  | Required | Cardinality | Max character size (English / translated)                                                                                                                              |
| ------------------------------------------------------------------------------------------ | ----------- | -------- | ----------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Paragraph                                                                                  | Long copy   | Yes      | 1           | LL (200 / 260)                                                                                                                                                         |
| Source (ex. name)                                                                          | Short copy  | No       | 1           | XS (65 / 85)                                                                                                                                                           |
| Copy 1 (ex. role)                                                                          | Short copy  | No       | 1           | XS (65 / 85)                                                                                                                                                           |
| Copy 2 (ex. company)                                                                       | Short copy  | No       | 1           | XS (65 / 85)                                                                                                                                                           |
| Quotation mark type                                                                        | Option list | No       | –           | Double curved (default) (“”) <br/>Single curved (‘’) <br/>Double angle («») <br/>Single angle (‹›)<br/>Low high reversed double curved („ “)<br/>Corner bracket (「」) |
| [CTA (text)](https://www.ibm.com/standards/web/carbon-for-ibm-dotcom/components/cta/#text) | Component   | No       | 1           | –                                                                                                                                                                      |
