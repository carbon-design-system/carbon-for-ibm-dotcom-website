## Default
The Callout &mdash; quote pattern uses high contrast colors to create a delightful pause in the narrative and create a powerful impression on the user.

![Default](../../images/pattern/callout-quote/callout-quote.jpg)

## Content guidance

| Field | Field type | Required   | Cardinality | Max character size (English / translated) |
| -------- | --------- | ---------- | ---------- | ----------|
| Paragraph | Long copy | Yes   | 1 | LL (200 / 260)|
| Source (ex. name)  | Short copy | No  | 1 | XS (65 / 85) |
| Copy 1 (ex. role)  | Short copy | No | 1 | XS (65 / 85) |
| Copy 2 (ex. company)  | Short copy | No | 1 | XS (65 / 85)   |
| Quotation mark type | Option list | No | - | Double curved <code>“”</code> (default) <br>Single curved <code>‘’</code><br>Double angle <code>«»</code><br>Single angle <code>‹›</code><br>Low high reversed double curved <code>„ “</code><br>Corner bracket <code>「」</code> |
| [CTA (link)](https://www.ibm.com/standards/web/carbon-for-ibm-dotcom/components/cta/) | Component | No | 1 | - |

To see the list of all character count standards, go to this <a href="https://github.com/carbon-design-system/carbon-for-ibm-dotcom-website/wiki/Character-count-standards" target="_blank" rel="noreferrer">link</a>.
