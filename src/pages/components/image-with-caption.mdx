## Default

The Image with caption supports images at 1x1, 2x1, 4x3, and 16x9 aspect ratios. please view the <a href="https://www.ibm.com/design/language/photography/tips-and-techniques" target="_blank">photography guideline</a> on how to choose and create an IBM Image.

## Content guidance

| Field                                                                                                             | Field type | Required | Cardinality | Max character size <br/>(English / translated) | Notes                                                                                                           |
| ----------------------------------------------------------------------------------------------------------------- | ---------- | -------- | ----------- | ---------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| Heading                                                                                                           | Short copy | Yes      | 1           | XXS (40 / 55)                                  | This field also acts as the caption                                                                             |
| Description                                                                                                       | Long copy  | No       | 1           | L (150 / 200)                                  | This is only shown in the lightbox media viewer                                                                 |
| [Image](https://www.ibm.com/standards/web/carbon-for-ibm-dotcom/components/image)                                 | Component  | Yes      | 1           | –                                              | –                                                                                                               |
| [Lightbox media viewer](https://www.ibm.com/standards/web/carbon-for-ibm-dotcom/components/lightbox-media-viewer) | Component  | No       | 1           | –                                              | If enabled, the user can view the image, heading, and description in the lightbox modal by selecting the image. |

To see the list of all character count standards, go to this <a href="https://github.com/carbon-design-system/carbon-for-ibm-dotcom-website/wiki/Character-count-standards" target="_blank" rel="noreferrer">link</a>.
