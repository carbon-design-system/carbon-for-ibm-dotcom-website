## Overview

The Horizontal rule provides thematic breaks or divisions between content. Its primary role is to visually separate content and create clear distinctions between sections of a web page. It can also be used as a graphical element to add structure or delineate pieces of information.

## Variations

### Fixed

In this variation, the length of the Horizontal rule does not change based on browser size, and there are several fixed sizes to choose from.

### Fluid

In this variation, the Horizontal rule resizes as the window size changes. The overhang and inset are the two main styles for this variation.
