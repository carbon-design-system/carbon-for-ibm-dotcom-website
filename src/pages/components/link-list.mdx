## Default

The Link list displays links in a card format, with a semibold heading and the icon shown below the link text. It is typically used for resource links on the side of the main page content.

## Variations

### Vertical

This variation displays links as a minimal, vertical list with an icon. It is typically used for resource links on the side of the main page content and features options to include a semibold heading or a horizontal rule at the bottom. The icons can be placed on the left or right side of the text.

### End of section

This variation displays links as a vertical list with a semibold heading, divided by a horizontal rule, where icons appear to the far right of the text. It is typically used at the end of a content section to list a number of CTAs for the user.
