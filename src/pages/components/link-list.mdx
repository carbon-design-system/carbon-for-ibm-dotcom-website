## Default
Default Link list displays links in a card format, with a semibold heading and  the icon shown below the link text. The default Link list is typically used for resource links on the side of the main page content.

## Variations

### Vertical
Vertical Link list displays links as a vertical list with a semibold heading, where icons can be placed on the left or right side of the text. This variaton is typically used for resource links on the side of the main page content.

![Image of Table of contents component](../../images/component/table-of-contents/table-of-content-with-heading-content.jpg)

<Title>Use case</Title>

![Image of Table of contents component](../../images/component/table-of-contents/table-of-content-with-heading-content-use-case.jpg)

### Vertical with cards
Vertical with cards Link list combines the Default and Vertical Link list. This variation is typically used for varying resource links on the side of the main content of the page.

![Image of Table of contents component](../../images/component/table-of-contents/table-of-content-with-heading-content.jpg)

<Title>Use case</Title>

![Image of Table of contents component](../../images/component/table-of-contents/table-of-content-with-heading-content-use-case.jpg)

### End of section
End of section Link list displays links as a vertical list with a semibold heading, divided by a horizontal rule, where icons appear to the far right of the text. This variation is typically used at the end of a section of content to list a number of CTAs for the user.

![Image of Table of contents component](../../images/component/table-of-contents/table-of-content-with-heading-content.jpg)
