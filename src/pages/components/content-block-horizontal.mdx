## Overview

On a page with largely vertical content, Content block horizontal presents a
moment to enhance vertical ryhthm by switch up the reading direction. It is also
a component good for presenting large amount of CTAs with context. Please note
that no CTA should be used within this Content block except the ones included
within
[content item horizontal](https://www.ibm.com/standards/carbon/components/content-item-horizontal)
components.

## Content guidance

| Field                                                                                                | Field type | Required | Cardinality | Max character size (English / translated) |
| ---------------------------------------------------------------------------------------------------- | ---------- | -------- | ----------- | ----------------------------------------- |
| Headline                                                                                             | Short copy | Yes      | 1           | XXS (40 / 55)                             |
| TOC headline override                                                                                | Short copy | No       | 1           | Mini (25 / 35)                            |
| [Content item – horizontal](https://www.ibm.com/standards/carbon/components/content-item-horizontal) | Component  | Yes      | 1-8         | –                                         |

To see the list of all character count standards, go to this
<a href="https://github.com/carbon-design-system/carbon-for-ibm-dotcom-website/wiki/Character-count-standards" target="_blank" rel="noreferrer noopener nofollow">link</a>.
