## Overview

Content item horizontal is ideal when heading and copy text are short. A horizontally-distributed column layout helps to reduce page height, and it provides vertical scannability when multiples of these components are stacked.

Don't use this component if your heading and/or copy text are lengthy. Too much copy in this component results in a squeezed and tall text block that increases page height and is uncomfortable to read.

![Content item horizontal default](../../images/component/content-item-horizontal/content-item-horiz-default.png)

## Variations

### Content item horizontal with media

This variation shows a similar layout and is capable of featuring one piece of media (image or video). Media is required for this component to work. Media can be positioned either left or right of the text content.

![Content item horizontal media](../../images/component/content-item-horizontal/content-item-horiz-media.png)

<Caption>Examples of content item horizontal with media in two different configurations</Caption>

### Content item horizontal with thumbnail

When imagery is secondary, use the content item horizontal with thumbnail. This is ideal for listing blogs and events where a thumbnail image is a nice-to-have but not essential to the message.

![Content item horizontal thumbnail](../../images/component/content-item-horizontal/content-item-horiz-thumbnail.png)

<Caption>Example of content item horizontal with thumbnail</Caption>
