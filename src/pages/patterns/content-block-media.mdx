## Default

By default the Content block &mdash; with media starts with a brief overview before going into one or more detailed points. At strategic points it allows for an optional piece of media to support its content.

![A visual of Content block &mdash; with media with its overview and supporting content.](../../images/pattern/content-block-media/content-block-media--default.jpg)

## With aside elements
This variant is the same as the default version except that it allows for a list of related navigational items off to the side.

![A visual of Content block &mdash; with media with optional navigational items off to the side.](../../images/pattern/content-block-media/content-block-media--aside.jpg)
