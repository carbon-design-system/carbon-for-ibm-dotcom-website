## Feature card block &mdash; medium
The Feature card block &mdash; medium pattern is used to present a group of information with a medium size card with an image.

![Feature card block medium](../../images/pattern/feature-card-block-medium.jpg)

## Feature card block &mdash; large
The Feature card block &mdash; large pattern is used to present main piece of content as a main feature.
